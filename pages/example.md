# 첫 번째 게시글

이것은 GitHub Pages 정적 블로그의 샘플 게시글입니다.

## 소개

이 블로그는 다음과 같은 기술로 구축되었습니다:

- HTML, CSS, Vanilla JavaScript
- marked.js (마크다운 파싱)
- Prism.js (코드 하이라이팅)
- Giscus (댓글 시스템)

## 코드 예시

다음은 JavaScript 코드 예시입니다:

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}

greet('World');
```

## 마크다운 기능

이 블로그는 다양한 마크다운 기능을 지원합니다:

- **볼드 텍스트**
- *이탤릭 텍스트*
- [링크](https://example.com)
- `인라인 코드`
